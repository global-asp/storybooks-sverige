Hur man använder sidan

Hur du använder den här sidan när du:

läser berättelserna

lyssnar på berättelserna

laddar ner berättelserna

Att läsa berättelserna

  • Klicka på en berättelse för att läsa på den.
  • Klicka på Byt språk för att läsa berättelsen på ett annat språk.
  • Klicka på sv och en ikonerna för att snabbt byta mellan t.ex. svenska och engelska.
    • Notera att detta bara ändrar språket i texten. Om du vill ändra språket i ljudfilen klickar du på Byt språk.

Att lyssna på berättelserna

  • Klicka på ikonen bredvid valfri text för att höra en uppläsning av texten.
    • För att pausa eller starta om ljudfilen när du vill klickar du på ikonen.
  • Klicka på ljudkontrollerna överst på sidan för att höra en uppläsning av berättelsen (notera att inte alla berättelser har ljudversioner.)
    • Om du inte ser några ljudkontroller, eller om ljudkontrollerna är gråmarkerade, så betyder det att det inte finns någon ljudfil för det språket som du har valt vid just det tillfället.

Att ladda ner berättelserna

  • Klicka på Ladda ner PDF knappen högst upp eller ner på sidan.
  • Högerklicka på ljudkontrollerna längst upp på sidan för att ladda ner en berättelses hela uppläsning. Välj Spara audio som... för att spara ljudfilen till din dator.
    • Beroende på ditt operativsystem kan det finnas en ikon för nedladdning på din kontrollpanel när du klickar för att spara berättelsen till din dator.
