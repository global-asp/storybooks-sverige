جَسَدِي Min kropp My body

Clare Verbeek, Thembani Dladla, Zanele Buthelezi

Mlungisi Dlamini, Ingrid Schechter

Heba Tesheh, Maaouia Haj Mabrouk

Mashael Muhanna

Arabic

Level 1

أَسْتَطِيعُ بِهِ أَنْ أَرْكُضْ.

Jag kan springa med den.

I can run with it.

أَسْتًطيعُ بِهِ أَنْ أَقْفِزْ.

Jag kan hoppa med den.

I can jump with it.

أَسْتَطِيعُ بِهِ َأَنْ أَرْقُصْ.

Jag kan dansa med den.

I can dance with it.

أَسْتَطِيعُ بِهِ أَنْ أَسْبَحْ.

Jag kan simma med den.

I can swim with it.

أَسْتَطِيعُ بِهِ أَنْ أَقْفِزْ.

Jag kan hoppa hopprep med den

I can skip with it.

أَسْتَطِيعُ بِهِ أَنْ أَرْكُلْ.

Jag kan sparka med den.

I can kick with it.

أَسْتَطِيعُ بِهِ أَنْ أَهْرُبْ.

Jag kan fly med den.

I can escape with it.

لَكِنَّنِي لاَ أَسْتَطِيعُ بِهِ أَنْ أَطِيرْ.

Men jag kan aldrig flyga med den.

But I can never fly with it.

Written by: Clare Verbeek, Thembani Dladla, Zanele Buthelezi
Illustrated by: Mlungisi Dlamini, Ingrid Schechter
Translated by: Heba Tesheh, Maaouia Haj Mabrouk
Read by: Mashael Muhanna
Language: Arabic
Level 1
Source: My body from African Storybook
