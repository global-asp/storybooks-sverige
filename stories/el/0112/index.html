Το σώμα μου Min kropp

Mlungisi Dlamini, Ingrid Schechter

Eleni Manou

grekiska

Nivå 1

Det finns ingen ljudfil till denna berättelse ännu.

Μπορώ να τρέξω με αυτό.

Jag kan springa med den.

Μπορώ να πηδήξω με αυτό.

Jag kan hoppa med den.

Μπορώ να χορέψω με αυτό.

Jag kan dansa med den.

Μπορώ να κολυμπήσω με αυτό.

Jag kan simma med den.

Μπορώ να χοροπηδήξω με αυτό.

Jag kan hoppa hopprep med den.

Μπορώ να κλοτσήσω με αυτό.

Jag kan sparka med den.

Μπορώ να ξεφύγω με αυτό.

Jag kan fly med den.

Αλλά δεν μπορώ να πετάξω με αυτό.

Men jag kan aldrig flyga med den.
