Nivå 1

Je peux courir avec.

Jag kan springa med den.

Je peux sauter avec.

Jag kan hoppa med den.

Je peux danser avec.

Jag kan dansa med den.

Je peux nager avec.

Jag kan simma med den.

Je peux sauter à la corde sauter avec.

Jag kan hoppa hopprep med den.

Je peux donner un coup de pied avec.

Jag kan sparka med den.

Je peux m’échapper avec.

Jag kan fly med den.

Mais je ne peux jamais voler avec.

Men jag kan aldrig flyga med den.
