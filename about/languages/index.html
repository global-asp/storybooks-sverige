Språk

The following languages are currently included in the scope of the Sagor för barn på svenska project:

  • svenska
  • finska
  • meänkieli
  • samiska (nordsamiska)
  • romska (romani chib / zigenska)
  • jiddisch
  • arabiska
  • bulgariska
  • bosniska
  • danska
  • engelska
  • franska
  • grekiska
  • italienska
  • kroatiska
  • spanska
  • japanska
  • kinesiska (mandarin)
  • kurdiska (sorani / sydkurdiska / centralkurdiska)
  • montenegrinska
  • norska (bokmål)
  • persiska
  • polska
  • ryska
  • serbiska
  • turkiska
  • tyska
  • älvdalska (älvdalsmål)
  • gutniska (gutamål / gotländska)
  • jämtska (jämtmål)
  • skånska

Please note that some translations and recordings are not yet complete.

We are always looking for translators! If you speak one of the above languages and would like to help us translate or record stories, please contact us at globalafricanstorybook [at] gmail [dot] com
